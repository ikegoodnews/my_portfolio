Front-end Developer

What you need to know to be a Front-end Developer

Front-end developer is everything you see and interract with in the browser as a user or client of the application. It is a generic term for someone who builds the user interfaces (UI) and parts of an application using HTML, CSS and JavaScript.
Front-end development tools are text editor (VS Code, Atom, Sublime Text), browser development tools (Chrome), version control (Git, GitHub).
Other important technologies to learn are SASS/CSS Preprocessors, JavaScript Libraries like jQuery, JavaScript frameworks and libraries like React, Angular and Vue.Js, JavaScript framework like BootStrap, Command Line/CLI like Terminal, WordPress, Image Editors like Sketch, Photoshop, Wireframe tools like Figma, Webflow and Balsamiq.
Resources for learn are Podcasts, Video tutorials, Books, Applications, Conferences, Meetups, w3school, etc.

Full Stack Developer

What you need to know to be a Full Stack Developer

Full Stack Developer is someone who is skilled in both the front-end and back-end technologies.

Back-end Developer

What you need to know to be a Back-end Developer

Back-end Developer - or Serve Side Application - is where the application data is saved, updated, manipulated and sent back to the client side. Learn languages like Java, JavaScript, Python, Ruby, and PHP, which stores and retrieve user data or securing accepting and storing credit card numbers. Back-end developer must be familiar with web servers like APACHE, NGINX, Microsoft IIS (IIS means Internet Information Services). Web frameworks help back-end developers in programming. JavaScript frameworks are Node.Js and Express, Java frameworks are Spring and JSF (JSF means JavaServer Faces) - Java Is not the same as JavaScript, Python frameworks are Django and Flask, C# frameworks are ASP.NET MVC, ASP.NET CORE. Other technologies to be familiar with are SQL, API like JSON, Database software like MySQL, SQL Server, PostgreSQL. Version Control Softwares like GIT, Subversion. Cloud Hosting platforms like Amazon Web Services (AWS), Heroku and Google Cloud Platform.
